# Пояснительная записка — структура

Тема: Игра «Судоку» с голосовым управлением.

## Аннотация

Курсовая работа посвящена разработке мультимодального приложения «Судоку голосом» для ассистентов Салют. Приложение позволяет решать головоломку Судоку с помощью экранного интерфейса и голосовых команд.

## Введение

Судоку — логическая головоломка на поле 9×9. Цель игры — заполнить пустые клетки цифрами от 1 до 9 так, чтобы в каждой строке, столбце и квадрате 3×3 цифры не повторялись.

## Постановка задачи

Необходимо реализовать Canvas App с игровым полем, генерацией задач, проверкой решения и голосовым управлением.

Функции:
- запуск новой игры;
- выбор сложности;
- генерация задачи с единственным решением;
- выбор клетки;
- ввод цифры;
- очистка клетки;
- подсказка;
- проверка поля;
- определение победы;
- голосовое управление всеми основными действиями.

## Описание проектных решений

Проект состоит из двух компонентов:

1. Front-end Canvas App на React.
2. Голосовое приложение SmartCode/Sber.

Backend отсутствует, так как проект не требует серверного хранения данных.

Алгоритм генерации:
1. Формируется полностью решенное поле Судоку.
2. Выполняются случайные перестановки строк, столбцов, блоков и цифр.
3. Из поля удаляются числа в зависимости от сложности.
4. После каждого удаления запускается solver с подсчетом решений.
5. Если решений становится больше одного, удаление отменяется.
6. В результате получается задача с единственным решением.

## Заключение

В результате работы было разработано приложение «Судоку голосом». Реализована генерация задач, экранное управление, проверка ошибок, подсказки и обработка голосовых команд.
