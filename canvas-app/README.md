# Судоку голосом — Canvas App

Фронтенд для смартаппа Салют: игра Судоку 9×9 с экранным и голосовым управлением.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте адрес из терминала, обычно `http://localhost:5173`.

## Сборка

```bash
npm run build
npm run preview
```

Папка `dist` — собранный фронтенд для размещения на хостинге.

## Основные команды

- Новая игра
- Новая лёгкая игра
- Новая средняя игра
- Новая сложная игра
- Выбери строку 3 столбец 4
- Поставь 5
- Поставь 5 в строку 3 столбец 4
- Очисти клетку
- Подсказка
- Проверить
- Что ты умеешь
