# Судоку голосом

Готовый учебный проект для курсовой работы по клиент-серверным приложениям.

Состав:

- `canvas-app/` — фронтенд Canvas App на React/Vite.
- `smartcode/` — заготовка голосового приложения SmartCode/Sber.
- `docs/` — материалы для пояснительной записки.

Backend отсутствует: состояние игры хранится во фронтенде, а голосовой сценарий отправляет команды во фронтенд.

## Быстрый запуск фронтенда

```bash
cd canvas-app
npm install
npm run dev
```

## Сборка фронтенда

```bash
npm run build
```

## Проверка без Салюта

Внизу интерфейса есть поле «Тест голосовой команды в браузере». Введите туда текстовую команду, например:

- `выбери строку 3 столбец 4`
- `поставь 5`
- `поставь 5 в строку 3 столбец 4`
- `подсказка`
- `проверить`
- `новая сложная игра`

## Что важно доделать в SmartMarket Studio

Фронтенд уже готов к приему команд формата JSON:

```json
{ "type": "SET_NUMBER", "number": 5, "row": 2, "col": 3 }
```

В SmartMarket Studio нужно создать интенты с сущностями `number`, `row`, `col` и отправлять во фронтенд соответствующие действия.
